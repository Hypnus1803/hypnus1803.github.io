# Personal Webpage

This is my personal webpage hosted on GitHub Pages. The public entry point is https://hypnus1803.github.io/JoseIvan/ and the bare root URL intentionally serves the custom 404 page. A separate section for selling and giving away items lives at https://hypnus1803.github.io/gone_soon/. The site contains sections for my profile, interests, and Python for Solar Physics.

## Structure

- JoseIvan/: Public entry point and container for the whole public site.
- JoseIvan/profile/: Contains profile-related pages.
- JoseIvan/interests/: Contains interests-related pages.
- JoseIvan/solar_python/: Contains Python for Solar Physics-related pages.
- JoseIvan/css/: Contains styles for the website.
- gone_soon/: Independent minimalist pages for selling items and giving things away.
